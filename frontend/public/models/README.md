# Face-api.js Models Setup

This directory contains the pre-trained models required for face detection and recognition.

## Files

- `tiny_face_detector_model-*` - Lightweight face detection model
- `face_landmark_68_model-*` - 68-point facial landmark detection
- `face_recognition_model-*` - Face descriptor extraction for matching

## Downloaded from

Official face-api.js repository: https://github.com/justadudewhohacks/face-api.js

## Total Size

~6.8 MB

## Usage

These models are automatically loaded by the face matching utility when users capture selfies during KYC verification.
